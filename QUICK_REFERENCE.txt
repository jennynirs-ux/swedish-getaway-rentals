HOW IT WORKS:
-------------
1. Browser requests page (e.g., GET /)
2. Next.js server runs async component
3. Server fetches data from Supabase
4. Server generates metadata (og:image, title, etc.)
5. Server renders to HTML with data
6. Browser receives complete HTML + JS
7. React hydrates for interactivity

WHAT YOU GET:
-------------
• Faster page loads (data in HTML, not fetched by browser)
• Better SEO (full metadata in HTML for crawlers)
• Social sharing works (og:image pulled from Supabase)
• 30-50% faster First Contentful Paint
• All existing functionality preserved

TROUBLESHOOTING:
----------------
Content not appearing?
  → npm run next:build (check for errors)
  → Check .env has VITE_SUPABASE_URL

Metadata not showing?
  → View page source (Ctrl+U / Cmd+U)
  → Check <head> for <meta> tags

Images not loading?
  → Check Supabase URLs are HTTPS
  → Check next.config.mjs remotePatterns

Slow builds?
  → Reduce generateStaticParams() limit (currently 50)
